# Pages Directory

This directory contains all page components in the application.

## Available Pages

1. **AIAssistantPage** - Main AI Assistant interface
   - Location: `src/features/AIAssistant/pages/AIAssistantPage.tsx`
   - Route: `/` or `/ai-assistant`

2. **Settings** - Application settings page
   - Location: `src/features/settings/pages/Settings.tsx`
   - Route: `/settings`

All pages are lazy loaded in `src/App.tsx` and routes are configured in `src/routes.ts`.
